# k8s-kuburnets-
Kubernetes（简称 K8s）是由 Google 发起的开源 容器编排平台，用于自动化部署、扩展和管理容器化应用。Docker 管容器，K8s 管很多 Docker 容器。

一句话

Kubernetes 是容器编排平台，具备自动部署、弹性伸缩、服务发现、负载均衡、故障自愈和滚动更新等能力，广泛应用于微服务、云计算平台、AI推理服务以及企业数字化系统的部署与运维。

K8s）是一个开源容器编排系统，可自动执行 容器化应用程序的部署、扩展和管理。

它将容器 分组为逻辑单元（Pod），并处理关键操作，例如自愈、负载均衡和 跨服务器集群的自动部署。

工作原理
Kubernetes 集群由控制平面（大脑）和多个工作节点（运行应用程序的机器）组成。
控制平面：做出 全局决策（例如，调度），检测并 响应集群事件。             （主要逻辑）
工作节点：实际运行 容器化应用程序 的虚拟机或物理计算机。

核心概念
Pod： K8s 中最小的 可部署单元，代表 集群中正在运行的  进程的单个实例。
部署：一个确保Pod正常运行  并  定义如何更新或扩展它们的对象。
服务：定义一组逻辑Pod的 抽象概念，支持 外部流量路由或内部通信。

主要优势
自动扩展：根据 资源利用率或自定义指标 自动扩展或缩减 应用程序。   （增添应用程序）
自我修复：重启发生 故障的容器，在节点死亡时替换并 重新调度容器，并终止对健康检查 无响应的容器。
可移植性：可在公有云（公有云服务器）、本地数据中心 和混合环境中运行，让您可以在任何地方 运行应用程序。
